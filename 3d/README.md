# Pocket Stand Keychain - Minimal

Soporte minimalista para movil tipo llavero, optimizado para impresion 3D FDM.

## Concepto

Diseno compacto y elegante para llevar en llaves o mochila:

- Forma limpia, sin ornamentacion excesiva.
- Ranura inclinada para apoyar el movil en vertical u horizontal.
- Tope frontal para evitar deslizamientos.
- Agujero reforzado para anilla de llavero.
- Alivios interiores para reducir peso sin perder rigidez.

## Medidas base

- Largo total: 70 mm
- Ancho: 28 mm
- Grosor: 5.2 mm
- Diametro agujero llavero: 5 mm
- Espesor minimo estructural: 2.4 mm

Compatible con la mayoria de telefonos (incluidas fundas finas/medias).

## Parametros que puedes ajustar rapido

- `body_len`, `body_w`, `body_h`
- `slot_angle` (angulo de vision)
- `slot_width` (fundas mas gruesas)
- `lip_h` (retencion frontal)
- `ring_hole_d` (tamano de anilla)

## Ajustes recomendados de impresion

- Material: PLA o PETG
- Altura de capa: 0.20 mm
- Perimetros: 4
- Relleno: 25% (Gyroid o Grid)
- Soportes: no
- Orientacion: plano sobre la cara inferior
